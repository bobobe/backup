nohup python preprocess.py 1>py.out 2>py.out &
(将标准输出1，和标准错误2都输出到py.out文件中） &是让保持后台运行，nohup是不挂断的运行命令，一般两个一起使用。
也可以简写为：
nohup python preprocess.py >py.out 2>&1 &
(标准输出1可以省略，标准错误2输出到标准输出1的文件地址）

nohup python train.py -gpus 0 1>py.out 2>py.out &
python predict.py -gpus 0 -restore checkpoint -log log_name
